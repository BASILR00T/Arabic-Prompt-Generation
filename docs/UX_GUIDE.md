# UX Guide

## Landing -> Workspace
1. Click **ابدأ مشروعك الآن** to open the Workspace.
2. Describe your idea in Arabic in the input box.
3. Pick:
   - **Mode**: Coding (Pro), Image Generation, Creative Writing, General
   - **Goal**: implementation, logic-fix, optimize
   - **Focus Area**: Logic, UI, Security, Optimization, Testing, Architecture
4. Click **إصدار الأمر البرمجي المعزز**.

## Workspace layout (no long scroll)
- The Workspace has a panel toggle:
  - **الإدخال**: show requirements + mode settings (Quick Templates included).
  - **الإخراج**: show the generated prompt + Arabic logic summary.
- After generating successfully, the UI automatically switches to **الإخراج**.

## Output panels
- **English Prompt Terminal (LTR / mono)**: the generated prompt you can copy.
- **Arabic Logic Summary**: a brief Arabic verification summary shown under the prompt.

## Tips for best results
- Include concrete constraints (framework, target runtime, performance goals).
- Mention what to avoid (e.g. “no magic shortcuts”, “secure by design”, “add tests”).
- Pick a Focus Area that matches your intended quality target.

## Mobile route
For a thumb-friendly layout, open the Workspace using:
- `/mobile`
