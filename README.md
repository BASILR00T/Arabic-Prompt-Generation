
# 🌌 Arabic Prompt Engineering Laboratory

A state-of-the-art, premium web application designed for elite Arabic Prompt Engineering. Built with **React 19**, **Vite 6**, and **Gemini AI**, this platform transforms architectural logic into high-performance AI instructions.

## ✨ Features

- **💎 Elite Glassmorphism UI**: A stunning, OLED-dark aesthetic optimized for focus and elegance.
- **🌍 Native Arabic Support**: Deeply integrated RTL support with professional **Tajawal** typography.
- **🧙‍♂️ Logic Transformation**: Transforming basic thoughts into structured, high-context AI prompts.
- **🏗️ Architecture Studio**: A dual-pane workspace for designing and reviewing prompts side-by-side.
- **⚡ Ultra-Refined Animations**: Framer Motion powered transitions for a seamless user experience.

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone <repository-url>
cd Arabic-Prompt-Generation
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Configure Environment Variables
Create a `.env.local` file in the root directory and add your **Gemini API Key**:
```bash
VITE_GEMINI_API_KEY=your_gemini_api_key_here
```

### 4. Start Development Server
```bash
npm run dev
```

## 📜 Professional Methodology

The laboratory employs a **Multi-Stage Refinement Architecture**:
1. **Context Extraction**: Analyzing the user's base intent.
2. **Logic Expansion**: Applying chain-of-thought and architectural patterns.
3. **Structure Compilation**: Formatting into a professional, high-precision prompt.
4. **Final Optimization**: Reviewing for clarity, tone, and performance.
